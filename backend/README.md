# Smart Darshan System

An AI-driven smart temple management and crowd control system
for temples in Vrindavan and Mathura.

## Features (Planned)
- Free darshan booking (first come first served)
- Limited slot capacity
- 2-month cooldown per user
- Live crowd monitoring
- Temple history & parikrama maps
- Hotels, parking, prasad services
- Admin & governance dashboard

## Tech Stack
- Backend: Node.js, Express
- Frontend: Next.js (planned)
- Database: PostgreSQL
- AI: TensorFlow.js
